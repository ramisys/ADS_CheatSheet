3. Views

Creating a View

CREATE VIEW employee_view AS
SELECT name, salary FROM employees WHERE salary > 3000;

Altering / Dropping Views

ALTER VIEW employee_view AS
SELECT name, salary, dept_id FROM employees;
DROP VIEW employee_view;

Updatable Views

UPDATE employee_view SET salary = 6000 WHERE name = 'John';

WITH CHECK OPTION

CREATE VIEW high_salary AS
SELECT * FROM employees WHERE salary > 5000
WITH CHECK OPTION;

Benefits

  • Simplifies complex queries
  • Improves security by limiting access
  • Provides logical data independence
